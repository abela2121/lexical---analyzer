# Web-Based Lexical Analyzer

A Flask web app that converts source code into tokens with line/column tracking.

## Features
- Tokenizes: KEYWORD, IDENTIFIER, NUMBER, OPERATOR, DELIMITER, ERROR
- Tracks exact line and column numbers
- Handles `.txt`, `.py`, `.cpp`, `.docx` files
- Never crashes on invalid input

## How to Run Locally
1. Install Python 3.8+
2. Clone this repo: `git clone https://github.com/YOUR_USERNAME/lexical-analyzer.git`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
5. Open `http://127.0.0.1:5000` in your browser
